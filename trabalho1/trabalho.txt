legenda:
-1 instruções para uso
-2 particularidades

1: 
	-ao iniciar o executavel, havera aplicativos iniciais na tela para
fins demonstrativos, e abaixo um menu com opções:

-next -> ira para a tela posterior, 	não hávera uma opção de next para a ultima

-ant -> ira para a tela anterior,	não hávera uma opção de ant para a primeira tela

-view -> mostrara os aplicativos a tela onde esta navegando

-rem -> ira perguntar um nome de aplicatiivo e remove-lo 
tomando as devidas medidas,	caso não há aquele aplicativo uma mensagem será exibidad

-busca ->pergunta um nome e apresenta se ele existir, a tela onde esta

-add ->	adiciona um aplicativo pelo nome, não sera possivel adicionar um já existente

2:
-ao clicar em next ou em ant mudara um ponteiro que não é o HEAD para a tela onde 
sera navegada, o HEAD servira para garantir que sempre tera uma tela principal
que podera até estar vazia mas nunca será deletada pelo própio software

-o número máximo de elementos em uma tela é 6, para facilitar a visualização
de novas telas sendo criadas

-não é possível adicionar elementos já existentes, caso tente isto, será 
exibido um alerta
 
-um contador para cada tela marca quantos aplicativos existentes
há na tela, inicialmente é 1 quando é criada uma nova e n para a principal
que ira vir com já com alguns aplicativos

-quando o contador chegar em 7 sera alocado dinamicamente um novo nó para
que será um tela que contera o ultimo elemento inserido, assim o contador na anterior 
volta a ser 6 e nesta nova tela inicia em 1

-se chegar em 0 este nó tela sera liberado da memória se e somente se não
for a tela principal, !=HEAD
